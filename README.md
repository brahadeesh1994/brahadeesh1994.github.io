# Academic Homepage

Source code for my academic website, hosted at https://brahadeesh1994.github.io.

## Tools

- HTML and CSS (no build step)
- MathJax 3 (loaded from CDN, for LaTeX rendering)
- Google Fonts: Roboto Mono
- Latin Modern Roman (loaded from CDN)

## Structure

- `index.html` — Home: bio, CAT Lab, news, contact
- `research.html` — Research interests
- `publications.html` — Publications, with collapsible abstracts
- `teaching.html` — Teaching record (instructor, TA, mentoring, supervision)
- `activities.html` — Talks, events organized, professional service, awards, institutional service
- `resources.html` — Reviews, hard-to-find papers
- `blog.html` — Blog index; individual posts in `blog/`
- `style.css` — Single shared stylesheet
- `assets/` — PDFs (CV, slides, posters, accepted manuscripts)
- `k7-torus.png` — Favicon and CAT Lab logo
